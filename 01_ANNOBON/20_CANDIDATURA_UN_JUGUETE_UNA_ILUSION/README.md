# README del proyecto

## Propósito del proyecto

Este dossier reúne la documentación de preparación y presentación del proyecto "Un Juguete, una Ilusión", orientado a la candidatura de colaboración para la infancia de Annobón, en la República de Guinea Ecuatorial.

El objetivo de este repositorio es centralizar la información necesaria para comprender, revisar y presentar el proyecto de forma coherente, organizada y trazable.

## Organización general del proyecto

El proyecto está organizado en torno a cuatro grandes bloques:

1. Documento principal de candidatura
2. Anexos temáticos de apoyo
3. Referencias documentales
4. Recursos visuales y materiales de apoyo

Esta estructura permite separar el contenido de presentación del contenido de trabajo interno y de apoyo documental.

## Estructura actual de los archivos

La carpeta principal contiene los documentos de base del proyecto, así como las carpetas de apoyo:

- Documentos principales del proyecto
- Anexos
- Referencias
- Imágenes
- Archivos de seguimiento y auditoría

## Convenciones de nombrado

Para mantener la coherencia documental, se recomienda respetar las siguientes reglas:

- Los nombres de los archivos se escriben en mayúsculas y con guiones bajos cuando es necesario.
- Los nombres deben ser descriptivos y orientados a la función del documento.
- Los documentos principales y anexos utilizan prefijos claros como "ANEXO_" o el nombre del documento central.
- Los archivos de referencia se agrupan por temática.
- Los recursos visuales se nombran de forma explícita y homogénea.

## Flujo de trabajo utilizado

El flujo de trabajo actual sigue esta lógica:

1. Elaboración del documento principal de candidatura.
2. Complementación con anexos temáticos.
3. Recolección de referencias de contexto y de apoyo.
4. Organización de imágenes y materiales visuales.
5. Uso de documentos de auditoría y checklist para revisar el avance del dossier.

Este enfoque permite trabajar de forma progresiva y mantener un seguimiento de los diferentes componentes del proyecto.

## Buenas prácticas para mantener el proyecto

Para conservar un nivel de organización profesional, se recomienda:

- Mantener los archivos en la carpeta correspondiente a su naturaleza.
- No mezclar documentos de presentación con documentos de trabajo interno.
- Actualizar este README y el INDEX cuando se agregue o modifique contenido.
- Usar nombres estables y consistentes para nuevos archivos.
- Guardar las referencias y las imágenes en sus carpetas específicas.
- Mantener una lógica de orden y trazabilidad en los documentos.
- Revisar periódicamente el estado del dossier y los documentos de seguimiento.
